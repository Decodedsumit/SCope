# SCope: Visualization of large-scale and high dimensional single cell data

## Server

### Data Server (DS)

- Install
```
cd scope-server

# Dependencies
pip install -r requirements.txt

# Server
python setup.py develop
```

- Run (development version)
```
scope-server
```

### Client

- Install
```
cd scope-client
npm install
```

- Run (development version)
```
cd scope-client
npm run dev
```
